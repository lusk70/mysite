# ページ詳細設計ドキュメント

## 1. トップページ（ホーム）

### ページ情報
- **URL**: `/`
- **ページタイトル**: 「企業名 - 日本酒醸造」
- **メタディスクリプション**: 企業の簡潔な説明

### ページ構成要素

#### 1-1. ヒーロー セクション
```
┌─────────────────────────────────┐
│     ヒーロー画像/動画           │
│   （醸造所外観または造りの様子） │
│                                 │
│     企業名（ロゴ）              │
│     キャッチコピー              │
│                                 │
│     [詳しく知る] ボタン         │
└─────────────────────────────────┘
```

#### 1-2. 特徴紹介セクション
- 3～4個のハイライト
  - 地域の伝統
  - こだわりの素材
  - 独自の製造技術
  - 他（例：受賞歴、認定など）

#### 1-3. 主要コンテンツへのナビゲーション
```
┌──────────────────────────────────────────┐
│  会社概要      思いを語る    アクセス    │
│  ┌────────┐   ┌────────┐   ┌────────┐   │
│  │概要link│   │思い link│   │MAP link│   │
│  └────────┘   └────────┘   └────────┘   │
└──────────────────────────────────────────┘
```

#### 1-4. オンラインショップへの導線
```
┌──────────────────────────────────┐
│  取り扱い商品の簡単紹介          │
│  商品画像のカルーセル            │
│                                  │
│  [ショップへ行く] ボタン        │
│  （外部リンク）                  │
└──────────────────────────────────┘
```

#### 1-5. フッター
- 企業情報
- ナビゲーションリンク
- SNSリンク
- 著作権表記

## 5. オンラインショップへのリンク

### 実装方法

#### パターン1: 同一サイト内のショップページ
- URL: `/shop` または `/store`
- 商品一覧表示
- 購入機能（決済システム連携）

#### パターン2: 外部リンク
- 既存ECプラットフォームへのリンク
- トップページに「ショップへ行く」ボタンを配置
- 各ページのCTA（Call To Action）として配置

### CTA（行動喚起）の配置例
- トップページ: ヒーロー下部、商品紹介セクション
- 会社概要: 主要商品紹介セクション下
- 思いを語る: ページ下部
- アクセス: ページ下部

## レスポンシブデザイン考慮事項

- **モバイル（SP）**: 480px以上
- **タブレット（tablet）**: 768px以上
- **デスクトップ（PC）**: 1024px以上

各デバイスでナビゲーション、画像、テキストレイアウトを最適化
