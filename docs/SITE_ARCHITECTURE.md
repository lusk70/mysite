# サイト構造ドキュメント

## プロジェクト概要

**プロジェクト名**: mysite
**コンセプト**: 日本酒製造を行う街の醸造所の企業サイト
**目的**: 企業の認知度向上、商品販売、顧客とのコミュニケーション

---

## サイト全体図

```
mysite（企業サイト）
├── トップページ（ホーム）
├── 会社概要
├── 思いを語るページ
├── アクセス
├── オンラインショップへのリンク
└── 共通要素（ヘッダー、フッター、ナビゲーション）
```

---

## ページ構成

### 1. トップページ（ホーム）
- **URL**: `/`
- **目的**: サイト全体の入口、企業の第一印象
- **主な要素**:
  - ヒーロー画像/セクション（醸造所の風景など）
  - 企業キャッチコピー
  - 主要コンテンツへのリンク
  - 最新情報/ニュースセクション（オプション）
  - 特徴紹介セクション

### 2. 会社概要ページ
- **URL**: `/about`
- **目的**: 企業情報、信頼感の構築
- **主な要素**:
  - 企業名、設立年
  - 企業理念・ビジョン
  - 事業内容の説明
  - 主要商品の紹介
  - スタッフ紹介（オプション）
  - 企業のhistory/timeline

### 3. 思いを語るページ
- **URL**: `/philosophy` または `/message`
- **目的**: 経営陣の想い、企業の価値観を伝える
- **主な要素**:
  - 製造理念
  - 日本酒への想い
  - 地域への想い
  - こだわりポイント
  - 今後のビジョン

### 4. アクセスページ
- **URL**: `/access` または `/contact`
- **目的**: 醸造所への来訪情報、お問い合わせ
- **主な要素**:
  - 地図（Google Map埋め込み）
  - 住所
  - 電話番号
  - 営業時間
  - アクセス方法（電車、車、駐車場情報）
  - 見学受付情報（オプション）
  - お問い合わせフォーム（オプション）

### 5. オンラインショップへのリンク
- **URL**: 外部リンク
- **目的**: 商品販売への導線
- **主な要素**:
  - ショップへのCTAボタン
  - 取り扱い商品の簡単説明（トップページで）
  - 直販サイトまたはECプラットフォームへのリンク

## 共通要素

### ヘッダー
- ロゴ
- グローバルナビゲーション（5ページへのリンク）
- 言語切り替え（オプション）

### フッター
- 企業情報（名前、住所）
- サイトマップ
- ソーシャルメディアリンク（SNS）
- 著作権表記
- プライバシーポリシー（オプション）
- 利用規約（オプション）

## 情報設計の考慮事項

1. **ユーザー層**:
   - 既存顧客（リピート訪問）
   - 新規顧客（商品購入意欲層）
   - ファン層（企業の思いに共感）
   - 来訪者（醸造所見学希望）

2. **主な訪問目的**:
   - 企業情報の確認
   - 商品購入
   - 醸造所への来訪方法確認
   - 企業の思い・こだわりの理解

3. **デバイス対応**:
   - PC対応
   - スマートフォン対応（必須）
   - タブレット対応
